Бот для поиска аниме
1. Команда /search -> появляется кнопки по типу жанр, описание и студия допустим.
После нажатия на кнпоку можно написать условие, допустим для жанра то написать жанр и потом короче выводит первых 10 аниме.
2. команда info которая показывает инфу по аниме через имя
3. сделать список для проссмотра. В команде info будет кнопка добавить в список этот и будкт новая команда для проссмотра этого списка
